**SecureVault**

SecureVault is a secure file storage and user activity management system developed using Streamlit and Python. The application allows users to upload and manage files securely while tracking all user activities for transparency.

**Features**

• User signup and login system 

• Secure password hashing (SHA-256) 

• CAPTCHA based login protection 

• File upload, download, rename and delete 

• User-specific file storage 

• Activity logging with timestamp 

• Download activity log as PDF or CSV 

• Admin panel for viewing all user files 

• Light and Dark theme support

 **Technology Used**

• Python 

• Streamlit 

• SQLite 

• ReportLab (for PDF generation) 

• Custom CSS 

• Streamlit-extras (stylable_container)
